# 🚢 Titanic AI Dashboard

## Project Overview

This project is an interactive Titanic Analytics Dashboard developed using Streamlit and Python.

The dashboard allows users to upload a Titanic dataset and explore insights through KPIs, visualizations, data cleaning, and interactive filters.

## Features

* Dataset Overview
* Data Cleaning
* Missing Value Analysis
* KPI Dashboard
* Interactive Filters
* Survival Analysis
* Gender Distribution Analysis
* Passenger Class Analysis
* Fare Analysis
* Embarkation Port Analysis
* Age Distribution Analysis

## Technologies Used

* Python
* Streamlit
* Pandas

## Dataset

Titanic Dataset containing passenger information such as survival status, passenger class, gender, age, fare, embarkation details, and other passenger attributes.

## How to Run

### 1. Install dependencies:

```bash
pip install -r requirements.txt
```

### 2. Run the application:

```bash
streamlit run app.py
```

### 3. Upload the Titanic CSV file.

### 4. Explore the dashboard.

## Dashboard Components

### Dataset Overview

Displays dataset dimensions, column information, and sample records.

### Data Cleaning

* Removes duplicate records
* Handles missing values
* Standardizes dataset columns

### Key Performance Indicators (KPIs)

* Total Passengers
* Total Survivors
* Average Age
* Average Fare
* Number of Passenger Classes

### Interactive Filters

* Gender Filter
* Passenger Class Filter

### Visualizations

* Gender Distribution
* Survival Distribution
* Passenger Class Distribution
* Average Fare by Class
* Embarkation Distribution
* Age Distribution
* Passenger Category Analysis
